# Subular

Subular is a Angular 2 Subsonic/Madsonic music player.

after cloning the repository run in project directory

`npm install`

then

`npm start`


made with:
* AngularJS 2
* CryptoJS
* Color-Thief
* Bootstrap
* Typescript

You can download Sonic here:
http://www.subsonic.org/
